# Workflow Application

## Overview
This is a Spring Boot-based workflow management system that handles document upload, verification, and notifications using Camunda BPMN.

##Tech Stack
- Java 8
- Spring Boot
- Camunda BPMN
- Oracle Database
- REST APIs

## Features
- Upload documents
- Verify documents using workflow
- Automated notifications
- Multi-step approval process

## Architecture
- Microservice-based structure
- BPMN workflow integration
- Database-driven processing

## How to Run
1. Clone the repo
2. Open in IntelliJ
3. Configure database in `application.properties`
4. Run `WorkflowApplication.java`
